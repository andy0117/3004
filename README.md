##Using Git
First CD to where you want the local repo.

    git clone git@github.com:louisstow/3004.git
	
To add some files to the repo:

    git add .
	git add myFile.jsp
	
Commit:

    git commit -m "* My message regarding what changed"
	
Push to GitHub:

    git push
	
Pull the latest copy from GitHub:

	git pull
